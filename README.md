      PHP CRUD Application  

Overview 
A simple PHP & MySQL based CRUD (Create, Read, Update, Delete) application built using core 
PHP and custom CSS. This project demonstrates database connectivity, record management, and 
basic web application flow. 
Features - Add new clients - View all clients in a table - Update existing records - Delete records - MySQL database integration - Clean UI using custom CSS (no Bootstrap) - Simple & beginner-friendly structure 
Tech Stack - PHP - MySQL - HTML5 - CSS3 - XAMPP / Apache Server- Git & GitHub 

Project Structure 
CRUD/ 
■■■ index.php # List all clients 
■■■ create.php # Add new client 
■■■ edit.php # Update client 
■■■ delete.php # Delete client 
■■■ crud.sql # Database export file 
 

How to Run Locally 
1. Install XAMPP/WAMP/MAMP. 
2. Clone the repository and move it to xampp/htdocs/. 
3. Create database named crud in phpMyAdmin and import crud.sql. 
4. Configure DB credentials in PHP files. 
5. Run: http://localhost/CRUD/index.php
